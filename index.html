Quick example
hello.pscript
$/ pcript ex
lib import random
lib import random.number

list names = ["Alice", "Bob", "Charlie"]
int balance = 1000

for name with names;
  int bonus = number(from=10, to=100)
  int balance = balance + bonus
  print(name "got" bonus)
stop;

print("Final balance:" balance)
Why
🔒

Statically typed

Types are declared explicitly and resolved at compile time. No surprises at runtime.

📖

Readable aloud

Every line reads like a sentence. for fruit with fruits; — no cryptic symbols.

🧩

Library system

Install packages with pop get name from the Pop-Inc repository. Write your own .plib files.

🧠

Memory control

Set armemory=true at the top of your file to manage memory manually with alloc and free.
